Update Skelly

Skelly 1.0 comes with jQuery 3.6.0 and Bootsrap 5.1.3.

If you would like the latest jQuery

Right click the box below and save jquery.min.js over the file app/js/jquery.min.js

If you would like the latest Bootstrap

Right click the box below and save bootstrap.min.js over the file app/js/bootstrap.min.js

Right click the box below and save bootstrap.min.css over the file app/css/bootstrap.min.css

That's It!
